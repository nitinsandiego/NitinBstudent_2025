## âš™ï¸ Common Algorithms

### Sum all values:
```java
int sum = 0;
for (int r = 0; r < matrix.length; r++) {
    for (int c = 0; c < matrix[r].length; c++) {
        sum += matrix[r][c];
    }
}
```

### Count even values:
```java
int count = 0;
for (int[] row : matrix) {
    for (int val : row) {
        if (val % 2 == 0) count++;
    }
}
```
